# Support

## Getting Help

Use GitHub Discussions for usage questions, workflow ideas, and general help.

Use GitHub Issues for:

- reproducible bugs
- documentation problems
- specific feature requests

## Before Opening an Issue

- confirm you are using a current version from `main` or the latest release
- include your Python version, operating system, and Maya version
- include the exact tool call, error, and relevant logs
- state whether Maya `commandPort` is running locally

## Security Issues

For suspected vulnerabilities, follow [SECURITY.md](SECURITY.md) instead of
opening a public issue.
